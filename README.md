# 문장 순서 맞추기

영어 어구를 올바른 순서로 배열하는 모바일 대응 정적 웹앱입니다.

## 실행

`test` 폴더에서 로컬 웹 서버를 시작합니다.

```powershell
python -m http.server 8765
```

브라우저에서 `http://localhost:8765`를 엽니다. 휴대전화에서 확인할 때는 같은 네트워크에 연결한 뒤 `localhost` 대신 PC의 로컬 IP 주소를 사용합니다.

## 모바일 음성

- `audio/` 아래의 28개 WAV 파일을 직접 재생합니다.
- 휴대전화의 TTS 음성 설치나 언어 설정이 필요하지 않습니다.
- 모바일 브라우저의 자동 재생 정책을 지키기 위해, 지문 선택 또는 `다음 문장` 버튼을 누른 직후 음성이 시작됩니다.
- 일반 속도와 느린 속도 듣기를 지원합니다.

## 구성

- `index.html`: 앱 화면
- `styles.css`: 반응형 디자인 시스템과 화면 스타일
- `app.js`: 학습 진행, 채점, 재시도, 점수 저장, 음성 재생
- `audio/`: Reading 1·2 문장 음원
- `showcase.html`: UI 구성요소 확인용 페이지
- `DESIGN.md`: 디자인 토큰과 컴포넌트 규칙
